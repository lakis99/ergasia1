  • Οδηγός Git

Η εντολή $cd erg1 επιλέγει τον φάκελο τον οποίο έχουμε στον δίσκο μας.

Η εντολή $ git innit εγκαθιστά το repository που έχουμε δημιουργήσει στο site του github στο αρχείο μας.

Το $ git add . ενημερώνει τα αρχεία για να γίνει commit.

Το $ git commit -m "erg1" κάνει commit τα αρχεία που έχουν ενημερωθεί.

Το $ git push -u erg1 ανεβάζει τα αρχεία στο repository του github
